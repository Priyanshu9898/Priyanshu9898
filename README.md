## 🚀 What I Build

I build **production ML systems and agentic AI pipelines** that ship at enterprise scale.

🔷 **Brilliant Earth (MLE)** — Engineered Snowflake Cortex Agents across **783M+ records**, built an XGBoost conversion-prediction pipeline on **400M+ records with 700+ features** (↑27% accuracy), and designed **MCP-based orchestration** connecting Snowflake, Cortex Search, and Claude API via OAuth 2.0 — reducing analyst query prep time by 60%

🔷 **NC State Research** — Built **Lyra**, a cross-platform mental health app with an on-device **Qwen 2.5 LLM** (HIPAA-compliant), real-time **1,000Hz BLE sensor streaming**, and personalized **IHOPE ML models** exported via ONNX for on-device inference
